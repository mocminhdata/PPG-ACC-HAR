## Dataset

This study uses the **PPG-DaLiA** dataset, which includes synchronized wrist-worn sensor recordings from 15 subjects performing 8 daily activities: sitting, ascending/descending stairs, table soccer, cycling, driving, lunch break, walking, and working.

- **Source:** UCI Machine Learning Repository
- **URL:** https://archive.ics.uci.edu/dataset/495/ppg+dalia
- **DOI:** 10.24432/C53890
- **License:** CC BY 4.0
- **Device:** Empatica E4 wristband
- **Signals used:** PPG (64 Hz, downsampled to 32 Hz) and tri-axial ACC (32 Hz)

### Citation

> Reiss A, Indlekofer I, Schmidt P, Van Laerhoven K. Deep PPG: Large-scale heart rate estimation with convolutional neural networks. Sensors. 2019;19(14): 3079.
